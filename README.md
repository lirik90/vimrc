### Install
```shell
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```

### Install plugins
```
:PluginInstall
```

### Update (reinstall) plugins
```
:PluginInstall!
```

### Install YCM
```shell
cd ~/.vim/bundle/YouCompleteMe
python3 ./install.py --clangd-completer --go-completer --ts-completer
```
